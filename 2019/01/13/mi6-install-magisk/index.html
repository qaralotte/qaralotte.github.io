[MI6] 如何正确的安装Magisk

发布于 2019-01-13 Flash  Android MIUI Magisk 

Magisk，以通过启动时在 boot 中创建钩子，把 /data/magisk.img 挂载到 /magisk，构建出一个在 system 基础上能够自定义替换，增加以及删除的文件系统，所有操作都在启动的时候完成，实际上并没有对 /system 分区进行修改（即 systemless 接口，以不触动 /system 的方式修改 /system）

环境：Windows 10

手机上安装Magisk Manager

进入Magisk的Github
在Release里找到Magisk Manager最新的版本，这个是apk，可以在手机上直接安装
安装完之后它会叫你是否安装Magisk，点击仅下载，下载完之后它会告诉你Magisk下载的位置
记住它

当然，你直接在Github上下载Magisk是一样的，只不过你安装完Magisk你还得再安装Magisk Manager

检查及安装FastBoot驱动

请检查手机是否解BL锁 如未解BL锁 请前往官网申请解BL锁 否则进入不了FastBoot模式
解锁需双清
连接手机，手机关机之后按住音量减+电源键，之后会进入FastBoot模式
在cmd里执行代码

fastboot devices

如果说出现fastboot不是内部命令 也不是可运行程序，说明你的电脑没有安装fastboot驱动
那么请下载驱动
压缩包解压后
将adb.exe和fastboot.exe两个文件移到\Windows\System32里
将AdbWinApi.dll和AdbWinUsbApi两个文件移到\Windows\System里
这样fastboot驱动就可安装好了

刷入TWRP Recovery

官方的Recovery不能安装Magisk，这时就需要刷入一个第三方Recovery
在这里选择适用于本机型的TWRP
下载之后得到了一个.img文件，打开cmd，cd到这个.img父文件夹的位置
然后，开始刷入

fastboot flash recovery TWRP.img

完成之后重启

fastboot reboot

注意:执行重启命令后，则需一直按住音量加以进入TWRP Recovery，一旦重启至系统界面，这个TWRP Recovery会被管官方的Recovery覆盖，这个时候重启会直接进入官方的Recovery
如果你的手机已经重启至系统，请重复此步骤

开始刷入Magisk

成功进入TWRP Recovery，等待几秒种TWRP的准备工作
这时如果你的手机有锁屏密码，TWRP会要求你输入锁屏密码，若不输入，则无法载入/sdcard
设置TWRP语言为Chinese Simplified
接下来的操作就十分十分简单了
点击 安装
找到Magisk压缩包
安装
重启
Done

注意事项

  • 整个过程都不需要双清，也不需要解锁system分区，更不需要root
  • 若你之前已经安装了SuperSU，请删除，SuperSu与Magisk不兼容
  • 在安装插件以前，请安装Magisk Manager for Recovery Mode(mm管理器)，这个插件能在Recovery模式里删除插件

如何卸载Magisk

进入Recovery
点击 挂载
勾选data
返回到主界面
点击 高级-文件系统
进入/data/adb文件夹内，将这个文件夹完全删除
Done
